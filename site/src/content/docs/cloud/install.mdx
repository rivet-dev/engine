# Install Rivet CLI

Install the Rivet CLI to manage your projects from the command line.

## NPM Installation

To install globally:

<CodeGroup>
```sh {{"title":"Latest"}}
npm i -g rivet-cli@25.5.1
rivet --version
```
```sh {{"title":"Pinned Version"}}
npm i -g rivet-cli@25.5.1
rivet --version
```
</CodeGroup>

To run without install:

<CodeGroup>
```sh {{"title":"Latest"}}
npx rivet-cli@25.5.1
```
```sh {{"title":"Pinned Version"}}
npx rivet-cli@25.5.1
```
</CodeGroup>


## Binary Installation

### macOS & Linux & WSL

<CodeGroup>
```sh {{"title":"Latest"}}
curl -fsSL https://releases.rivet.gg/rivet/latest/install.sh | sh
rivet --version
```
```sh {{"title":"Pinned Version"}}
export RIVET_CLI_VERSION=25.5.1
curl -fsSL https://releases.rivet.gg/rivet/latest/install.sh | sh
rivet --version
```
</CodeGroup>

### Windows (PowerShell)

<CodeGroup>
```ps1 {{"title":"Latest"}}
iwr https://releases.rivet.gg/rivet/latest/install.ps1 -useb | iex
rivet --version
```
```ps1 {{"title":"Pinned Version"}}
$env:RIVET_CLI_VERSION = "25.5.1"
iwr https://releases.rivet.gg/rivet/latest/install.ps1 -useb | iex
rivet --version
```
</CodeGroup>

## Build From Source

```sh
git clone https://github.com/rivet-gg/rivet
cargo build --bin rivet
./target/debug/rivet --version
```
