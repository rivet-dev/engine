# Rivet Hub

[hub.rivet.gg](https://hub.rivet.gg)

## Developing

```bash
cd frontend/apps/hub
yarn install
yarn dev
```

Open [http://localhost:5080](http://localhost:5080) in your browser to view the website.

## Configuring

Create a file called `.env.local` to override properties.
