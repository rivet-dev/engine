# Limitations

## KV

| Feature                                    | Limit                                                     |
| ------------------------------------------ | --------------------------------------------------------- |
| **Key size**                               | 2 KiB                                                     |
| **Value size**                             | 128 KiB                                                   |
| **Keys per batch operation**               | 128                                                       |
| **Total payload size for `put` operation** | 976 KiB (includes keys and values together)               |
| **Total storage**                          | 1 GiB (per actor, [contact us](/support) for an increase) |
